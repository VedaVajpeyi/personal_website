Sociology x Product

Field Guide

A facilitation-first prototype. This version is designed less like a reading experience and more like something a team lead could use in a real session without needing extra translation.

If the right direction is “more usable in the room,” this is that argument.

Manual Workshop-first Facilitation
Current guide Book a workshop
01 Name the misread

Get the room to agree on what kind of problem this is before anyone reaches for a solution.

02 Run the models

Use the guide to test whether the issue is social, structural, metric, trust, or value-flow related.

03 Leave with one sentence

Write the diagnosis and first move in plain language before the meeting ends.

Sheet B Run The Seven Models
Model 1 The social adoption funnel

Use when the dashboard starts too late.

Model 2 Social risk vs. functional value

Use when usefulness is real but public use still feels costly.

Model 3 The behavior system

Use when the room cannot tell whether the break is person, culture, or system.

Model 4 Structure before motivation

Use when the desired action assumes time, permission, money, or slack users may not have.

Model 5 The metric is a social claim

Use when the KPI is improving but the lived product is deteriorating.

Model 6 The trust stack

Use when hesitation persists even though the interface is clear.

Model 7 Value flow

Use when growth depends on hidden labor, externalized cost, or off-dashboard harm.

Sheet C Meeting Script
1. Name the target behavior

What exactly do we need a user, team, or market to do?

2. Name the audience

Who is watching, approving, or withholding legitimacy?

3. Ask the time question

If this “works” for 12 months, what starts decaying underneath it?

4. Check the org mirror

Does our team structure reproduce the same problem we say users are facing?

Do not do first

Do not rewrite onboarding before checking whether the act is socially safe.

Do not do second

Do not add incentives before checking whether the behavior is structurally available.

Do not do at all

Do not skip the model that gives the room the uncomfortable answer.
